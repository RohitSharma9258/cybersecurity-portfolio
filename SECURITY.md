# Security Policy

## Supported Versions

Only the latest major version release is supported for active patch updates.

| Version | Supported |
| --- | --- |
| 5.x.x | Yes |
| < 5.0.0 | No |

## Reporting a Vulnerability

If you discover a security vulnerability in this portfolio, please do not open a public issue. Instead, email reports to **rohitsharma40421@gmail.com** with details on replication paths.

We will acknowledge receipt of the alert within 48 hours and work to provide a patch mitigation within 7 business days.
